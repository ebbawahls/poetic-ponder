# Poetic Ponder

A React and Typescript application that encourages you to read poetry and create your personal notes to each poem with a public API.

## API

The project uses two public APIs:

- PoetryDB
- DummyJSON

PoetryDB

GET:

- Fetch random poems
- Fetch one poem

DummyJSON

GET:

- Fetch all notes

POST:

- Create notes

PUT:

- Update notes

DELETE:

- Remove notes

## Vad innebär det att en webbplats är tillgänglig?

En webbplats som är tillgänglig är i sin design gjord så att alla användare kan ta del av innehållet, oavsett förutsättningar. Med hjälp av tydlig struktur, stöd för skärmläsare etc.

## Varför är tillgänglighet viktig vid utveckling av webbapplikationer?

Det gör applikationen en jämlik plats för så många som möjligt att ta del av information och funktionalitet.

## Ge tre konkreta exempel på hur ni kan göra en React-applikation mer tillgänglig.

1. Använda sig av semantisk HTML (h1, h2 etc)
2. Se till att färger har bra kontrast och läsbar text
3. Tillämpa sig av tangentbordsnavigation
